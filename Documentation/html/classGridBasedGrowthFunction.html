double *** GrowthMatrix
 The matrix of growth rates in (1/sec). It is a matrix of double triplets for growth rate at each grid point. The dimensions of the matrix are equal to (GridBasedGrowthFunction::nGridX, GridBasedGrowthFunction::nGridY), and set in constructor of the GridBasedGrowthFunction. The triplets store the growth rate in [ DV axis (x), AP axis (y), and AB axis (z)].

The constructor of GridBasedGrowthFunction.

The first six parameters will be directed to the parent constructor, GrowthFunctionBase::GrowthFunctionBase.
integers nX and nY will set GridBasedGrowthFunction::nGridX and GridBasedGrowthFunction::nGridY, respectively. GridBasedGrowthFunction::GrowthMatrix will be initiated to point at a 2 dimensional matrix of double triplets the size(nX, nY).
double*** GrowthMat is the pointer to the 2-dimensional matrix of double triplets, holding the 3D growth rates at each grid point. Values stored in GrowthMat will set the values in GridBasedGrowthFunction::GrowthMatrix. The matrix storing the growth rates have been read from an input file through ModelInputObject::readGrowthOptions and related functions therein

The function is to write the growth function summary to simulation summary file.

This function will write the GridBasedGrowthFunction details into the simulation summary file, provided as the first input. Time step (dt) of the simulation is provided as second input, to report the growth rates per hour. The output should look like:
Growth Type: growth From File (3) Initial time(sec): GridBasedGrowthFunction::initTime FinalTime time(sec): GridBasedGrowthFunction::endTime Growth matrix mesh size: GridBasedGrowthFunction::nGridX GridBasedGrowthFunction::nGridY
